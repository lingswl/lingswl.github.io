git使用教程

发表于2025-03-18|更新于2025-03-22|study
|浏览量:

1.git配置

代码存放的区域有四种，工作区，暂存区，本地，远程仓库

git config --list 查看配置信息

首先要配置邮箱和用户名

git config --global user.name " "

git config --global user.email " "

git config --global pull.rebase true

配置私钥和公钥

2. 初始化仓库

  • 创建本地仓库

git init 正常不使用这种方式，而是在远程创建一个仓库，然后pull过来

git status 查看状态

  • 添加文件到暂存区，以及撤回

git add . 添加所有文件

git add 文件 添加文件

git reset 文件 撤回文件

  • 修改commit 信息

没有添加到远程仓库 git commit –amend commit号会被修改

git push origin master -f 强制上传

已经添加到远程仓库

  • 产看远程仓库信息

git remote -v

git checkout 文件名 撤回修改 已经添加到远程仓库

git reset 撤回修改 没有添加到远程仓库

git reset –hard HEAD~ 撤回前一个commit

将仓库push到远程仓库

git push origin master
