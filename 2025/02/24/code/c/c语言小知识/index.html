1.void *

一般的32位系统指针都是占用4个字节，所以指针赋值仅仅只是这4个字节的赋值与类型没什么关系。

2.volatile 关键字

volatile 是来修饰易变变量的 ，编译器中的优化器在用到这个变量时必须每次都小心地从内存中重新读取这个变量的值，而不是使用保存在寄存器里的备份，有效的防止编译器自动优化，从而与软件设计相符合。
